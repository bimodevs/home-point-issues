# 🛠️ Home Point Issue Tracker

Welcome! Please use this space to report bugs or suggest new features. 

### 📋 Reporting Guidelines
To help fix issues faster, please follow this template:

1. **Environment:** Minecraft Version (e.g., 1.21.1) and Loader (Forge/Fabric/NeoForge).
2. **Mod Version:** (e.g., 1.4.0).
3. **Description:** What happened? What was expected?
4. **Logs:** Provide a link to [Pastebin](https://pastebin.com/) or [mclogs](https://mclo.gs/) with your `latest.log` or crash report.
5. **Steps to Reproduce:** * Type `/home`
   * Click `[Del]`
   * ...

---
**Note:** Please check if a similar issue already exists before opening a new one. Thank you for making **Home Point** better!
